# 🎯 AI Interview Preparation Platform

An intelligent and interactive web application designed to help students and job seekers prepare for technical interviews through AI-powered features, modern UI, and a seamless user experience.

## 🚀 Features

* 🤖 AI-powered interview preparation experience
* 💬 Interactive question and answer sessions
* 📚 Technical interview practice across multiple domains
* 🎨 Modern, responsive, and user-friendly interface
* 📱 Mobile-friendly design for learning anywhere
* ⚡ Fast performance with Vite and React
* 🔐 Authentication and backend integration with Supabase
* 📊 Clean architecture for scalability and maintainability

## 🛠️ Tech Stack

### Frontend

* React
* TypeScript
* Vite
* Tailwind CSS
* Framer Motion

### UI Components

* shadcn/ui
* Radix UI
* Lucide React

### Backend & Database

* Supabase

### Form Validation

* React Hook Form
* Zod

## ⚙️ Installation


Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
